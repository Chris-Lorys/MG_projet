# Move & Go — 
 MO_Conception de Sites Webs_ESIGELEC_25/26
 Plateforme Collaborative de Déménagement

## Fonctionnalités principales
- Authentification avec rôles (client / déménageur).  
- Création, consultation et gestion d’annonces.  
- Messagerie entre Clients et Déménageurs
- Upload de jusqu’à 10 images (JPEG, PNG, WebP).  
- Interface responsive basée sur Bootstrap 5.  
- Sécurisation des formulaires (CSRF, validation serveur, PDO).

## Technologies utilisées
- PHP 7 
- MySQL 
- HTML5, CSS3, Bootstrap 5
